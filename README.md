# 🎓 Student Record Management System

This project is a **Student Record Management System** built using **Python and Streamlit**.  
It allows users to add student records, view all students, search for a student, and see class statistics.

---

## 🚀 Features

- Add student details (Name, Roll Number, Marks)
- Calculate **Total, Average, and Grade**
- View all student records in a table
- Search student by **Roll Number**
- Display **Class Statistics**
- Show **Grade Distribution Chart**
